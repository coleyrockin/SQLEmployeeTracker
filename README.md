# SQLEmployeeTracker
Command-line application from scratch to manage a company's employee database, using Node.js, Inquirer, and MySQL.

### Usage and purpose
* This project was built using MySQL2 pkg, NODE.JS, Console.table pkg, Inquirer, and javascript.
